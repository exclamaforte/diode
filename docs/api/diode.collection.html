A class that hooks into the PyTorch feedback saver interface to collect matrix multiplication data with timing information and store it in a Dataset.

Supports two modes: 1. RANDOM: Generate random matrix shapes and collect data for specified operations 2. OPERATION_SHAPE_SET: Use predefined shapes from an OperationShapeSet
